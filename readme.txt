Indications permettant l'installation du projet : 
	- Utilisation de xampp (ou autre logiciel similaire) ==> activation d' Apache et MySQL
	- Importation de la base de données utilisée pour ce travail
	- Installation du composer via la commande " composer install " dans l'invite de 	commande. Cette étape doit-être réalisée à la racine du projet.
	- Configuration du fichier "conf.ini " situé dans " projetPHP/src/conf " pour l'adapter à 	PHPMyAdmin ou ses équivalents 
	- Exécuter le programme via l'URL de son navigateur en y entrant " localhost/projetPHP "
